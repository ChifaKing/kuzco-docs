# Kuzco Public API

## Desktop and CLI Latest Versions API
```
Endpoint: https://relay.kuzco.xyz/public/versions
Method: GET
```

### Description:

This endpoint returns the latest version information for both "desktop" and "cli" applications.

#### Response:
```
Status Code: 200 OK
Content-Type: application/json
```
### Usage:

To retrieve the latest version information, send a GET request to the /public/versions endpoint. The response will include the latest versions for both the "desktop" and "cli" applications.

#### Sample Request:
`curl -X GET https://relay.kuzco.xyz/public/versions`

#### Example Output:
```
{
  "desktop": "0.0.7-677c61c",
  "cli": "0.0.6-183e508"
}
```

## Hardware Information API
```
Endpoint: https://relay.kuzco.xyz/public/hardware
Method: GET
```

### Description:

This endpoint returns detailed information about the GPUs available on the network, including their types, VRAM, and share of the network.

#### Response:
```
Status Code: 200 OK
Content-Type: application/json
```

### Usage:

To retrieve the hardware information, send a GET request to the /public/hardware endpoint. The response will include details about the GPUs, total number of GPUs, and the total VRAM available.

#### Sample Request:
`curl -X GET https://relay.kuzco.xyz/public/hardware`
